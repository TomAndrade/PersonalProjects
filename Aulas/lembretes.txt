ESTE CURSO TRATARÁ APENAS EM TRêS TIPOS DE PROJETO: CONSOLE (DOS), DLLS E WINDOWS FORMS (GUI)

- Rebuild (recompilar) serve para compilar todo o código do zero enquanto o Build (compilar) irá compilar apenas o que foi alterado.

- Um namespace é uma forma de organizar e agrupar classes, interfaces, structs, enums e delegates dentro de um projeto. Ele funciona como uma "gaveta" onde você guarda tipos relacionados, evitando confusão e conflitos de nomes.

- Use Console.ReadKey() para aguardar que o usuário aperte alguma tecla. Geralmente usado para manter o executável aberto, como consoles.
- Use Console.ReadKey().KeyChar para guardar qual tecla foi apertada. Deve ser armazenada numa variável de mesmo tipo.
	- Para esconder a tecla que foi pressionada, insira "true": Console.ReadKey(true);

- Assim como no Java, use o \n para pular uma linha. Use \r\n para pular um a linha usando Windows. Apenas para o método Write.

- Delegates são semelhantes a uma variável que pode armazenar um método; trata-se de um tipo que faz referência a um método; similar a um callback em C/ C++; quando chamamos um delegate, todos os métodos que estão referenciados a ele serão chamados.
- Com delegates é possível criar callbacks, ou seja, uma variável que guarda um método que pode ser executado a qualquer momento;
- Com delegates conseguimos passar métodos como parâmetro para outros métodos
- Com delegates podemos chamar várias funcões com somente uma chamada de execução do delegate.
- O delegate sempre retornará o último método usado.

- Use o botão intervir durante uma depuração para analisar as ações que estão dentro do método, propriedade, estrutura ou variável.
- É possível arrastar a seta amarela da depuração para acessar determinados blocos do código sem passar por outros lugares antes.

- Classes internas são classes que só podem ser acessadas pelo assembly, ou seja, outros executáveis não podem acessá-las.
- Classes privadas só podem ser criadas dentro de outras classes.

- Use base() na assinatura de uma classe para aplicar um padrão de inicialização: public class Exemplo: base("1234"){}
	É como o método super() do Java.

-Eventos em C# são mecanismos que permitem que uma classe notifique outras quando algo acontece. Eles são usados para implementar o padrão publicador/assinante, muito comum em interfaces gráficas e sistemas reativos.

-Você precisa dos dois juntos porque:
	O delegate define como os métodos devem ser.
	O evento define quando e quem pode disparar ou se inscrever.

- Use os Timers para definir um tempo em que um determinado método será executado.

- WPF (Windows Presentation Foundation) - Framework que utiliza XAML para criar interfaces gráficas.
